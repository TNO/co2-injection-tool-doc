Managing the project
----------------------------------------------

Create a project
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
* Click New Project button
* A popup window will appear asking the project name
* Fill the project name
* Click OK

.. image:: images/newproject.png
    :width: 500


Open an existing project
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
* Click Open Project button
* A popup window will appear with a list of existing projects
* Select the project
* Click Open

.. image:: images/openproject.png
    :width: 500


Save a project
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
* Click Save Project button
* If project is not defined yet, it will ask the project name
* Project is saved

.. image:: images/saveproject.png
    :width: 500


Delete a project
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
* Click Delete Project button
* A popup window will appear asking to confirm
* Click OK

.. image:: images/deleteproject.png
    :width: 500
