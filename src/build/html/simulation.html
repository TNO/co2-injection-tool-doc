7. Running the simulation

7.1. How to set simulation setting

  • Fill in the simulation name

  • choose the start date

  • chose the end date

  • choose the interval of the simulation (hourly, daily, weekly, monthly, yearly)

  • select the scenario from dropdown list

7.2. How to run simulation

  • Make sure all the fields are filled and scenario has been chosen

  • Click Run Simulation Button

  • You can see the message in the console if the simulation has been started

7.3. How to check simulation status

  • The simulation is running in the background as a batch process

  • You can see the status by selecting the simulation name from the dropdown list

  • Click Get Status button
